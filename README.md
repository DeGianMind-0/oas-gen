# oas-gen

Just another OpenAPI code generator

## How to run

```shell script
java -cp <classpath> jsm.java.MainKt \
        -b <base-directory> \
        -s <schema-file> \
        -o <output-directory> \
        -p <namespace> \
        -g <generator-id> \
        -c <extra-schema-files>
```
