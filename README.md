# oas-gen

Just another OpenAPI code generator

## How to run

```shell script
java -cp <classpath> io.github.fomin.oasgen.java.MainKt \
        -b <base-directory> \
        -p <schema-path (relative to base directory)> \
        -s <schema-file> \
        -o <output-directory> \
        -n <namespace> \
        -g <generator-id>
```
